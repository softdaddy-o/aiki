## 한 줄 정의
API는 API(응용 프로그래밍 인터페이스)는 컴퓨터 간 또는 컴퓨터 프로그램 간의 연결입니다. 다른 소프트웨어에 서비스를 제공하는 일종의 소프트웨어 인터페이스입니다. 이러한 연결이나 인터페이스를 구축하는 방법을 설명하는 문서나 표준을 API 사양이라고 합니다. 이 표준을 충족하는 컴퓨터 시스템을 API를 구현하거나 노출한다고 합니다. API라는 용어는 사양이나 구현을 나타낼 수 있습니다라는 맥락에서 자주 언급된다.
## 어떻게 작동하나
API가 무엇인지, 기업에서 API를 사용하는 방법과 이유, AWS에서 API를 사용하는 방법을 알아보세요라는 설명을 함께 보면, API가 실제 제품과 연구 흐름에서 어떻게 쓰이는지 감이 잡힌다.
## 왜 지금 중요하나
AIKI 기사 기준으로 API는 21번 이상 함께 언급됐다. 그만큼 최근 AI 뉴스에서 맥락을 이해할 때 반복해서 마주치는 용어다.
