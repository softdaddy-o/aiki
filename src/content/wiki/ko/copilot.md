## 한 줄 정의
GitHub Copilot는 GitHub Copilot은 GitHub 및 OpenAI가 개발한 코드 완성 및 프로그래밍 AI 도우미로, 코드 자동 완성을 통해 Visual Studio Code, Visual Studio, Neovim, Eclipse 및 JetBrains 통합 개발 환경(IDE) 사용자를 지원합니다. 현재 개인 개발자와 기업이 구독을 통해 사용할 수 있는 생성 인공 지능 소프트웨어는 2021년 6월 29일 GitHub에서 처음 발표되었습니다. 사용자는 생성에 사용되는 대규모 언어…
## 어떻게 작동하나
GitHub Copilot은 편집기에서 직접 작업하여 전체 줄이나 전체 기능을 제안합니다라는 설명을 함께 보면, GitHub Copilot가 실제 제품과 연구 흐름에서 어떻게 쓰이는지 감이 잡힌다.
## 왜 지금 중요하나
AIKI 기사 기준으로 GitHub Copilot는 10번 이상 함께 언급됐다. 그만큼 최근 AI 뉴스에서 맥락을 이해할 때 반복해서 마주치는 용어다.
