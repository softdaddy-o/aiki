## 한 줄 정의
개발 생산성과 코딩 워크플로 작업에 쓰이는 AI 도구다. 쉽게 말하면 모델을 제품과 개발 흐름에 붙이는 작업대를 실제 제품과 워크플로로 옮긴 쪽에 가깝다.
## 실제로 무엇을 하나
모델 자체라기보다 개발 생산성과 코딩 워크플로 작업을 실제로 굴리는 도구 쪽에 가깝다. 모델 자체를 만드는 게 아니라 코드 작성, 검색, 수정, 실행, 리뷰 같은 반복 작업을 줄여 주는 층이다. 그래서 기능 목록보다 어떤 병목을 줄여 주는지로 읽는 편이 이해가 빠르다.
## 왜 중요한가
실제로는 모델 성능이 비슷할 때 생산성을 가르는 건 이런 도구와 워크플로인 경우가 많다. 실무에서는 도구 이름을 모델 이름처럼 오해하면 실제 도입 범위와 필요한 연결 작업을 잘못 보기 쉽다.
