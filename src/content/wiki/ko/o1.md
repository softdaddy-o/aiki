## 먼저 감 잡기
o1는 OpenAI가 제공하는 버전형 모델이다. 강화학습 기반 o-series reasoning 모델이다. 텍스트와 이미지 입력을 받아 긴 내부 추론 뒤 텍스트를 출력한다. 텍스트 입력·출력과 이미지 입력을 지원한다. 오디오와 비디오는 공식 미지원이다. 그래서 기사에서 이 이름이 보이면 추상적인 성능 향상 문구보다 입력 범위, 컨텍스트 한도, 접근 채널이 어떻게 달라졌는지부터 확인하는 편이 정확하다.
## 뉴스에서 왜 자주 나오나
o1는 AIKI 기사에서 이미 2번 이상 언급됐고, 기록도 2025-01-20까지 올라간다. 이제는 이름만 익히는 단계가 아니라 Introducing OpenAI o1 기준으로 Responses, Chat Completions, Realtime, Batch에서 쓸 수 있다. 파인튜닝과 distillation은 공식 미지원이다. OpenAI API 기준 1M 토큰당 입력 $15.00, 캐시 입력 $7.50, 출력 $60.00이다. 이 차이를 바로 읽어야 할 시점이다.
## 읽을 때 체크포인트
1. 먼저 o1가 어떤 입력을 받고 무엇을 출력하는지부터 확인하면 된다. 여기서 모델 포지션이 거의 정리된다.

2. 다음으로 컨텍스트, 최대 출력, 툴 호출 지원처럼 운영 조건을 봐야 한다. 같은 성능 홍보라도 실제 제품 적합성은 여기서 갈린다.

3. 마지막으로 Responses, Chat Completions, Realtime, Batch에서 쓸 수 있다. 파인튜닝과 distillation은 공식 미지원이다. OpenAI API 기준 1M 토큰당 입력 $15.00, 캐시 입력 $7.50, 출력 $60.00이다. 이 두 줄을 같이 읽으면 '당장 붙일 수 있는 모델인지'와 '비용이 감당되는지'를 빠르게 판단할 수 있다.
