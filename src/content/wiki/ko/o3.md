## 한 줄 정의
o3는 OpenAI가 어려운 추론, 계획 수립, 도구를 섞어 써야 하는 일 쪽 문제를 풀려고 내놓은 개별 모델 버전이야. 기사에서 이 이름이 보이면 상위 계열 소개가 아니라, 실제로 붙여볼 후보가 올라온 상황이라고 보면 돼. 텍스트와 이미지 입력을 받아 텍스트를 출력해. 오디오와 비디오는 기본 모델 기준 미지원이야. OpenAI의 상위 o-series reasoning 모델이야. 수학, 과학, 코딩, 시각 추론처럼 여러 단계를 거쳐야 하는 문제를 풀도록 설계됐어.
## 이 모델로 무엇을 할 수 있나
이 페이지에서 먼저 볼 건 "성능이 높다"보다 "어떤 일을 맡길 모델인가"야. OpenAI의 상위 o-series reasoning 모델이야. 수학, 과학, 코딩, 시각 추론처럼 여러 단계를 거쳐야 하는 문제를 풀도록 설계됐어. Responses API와 Chat Completions에서 바로 쓸 수 있고, Realtime·Assistants·Batch도 열려. 복잡한 추론 작업에 쓰는 상위 티어 API 모델로 보면 돼. 예를 들어 수학 풀이, 코드 수정, 장문 계획처럼 중간 단계가 긴 작업 후보를 고를 때 차이가 크게 난다. 그래서 어려운 추론, 계획 수립, 도구를 섞어 써야 하는 일처럼 한 단계씩 풀어야 하는 작업에 맞는지, 아니면 더 가볍고 싼 모델로도 충분한지 가르는 기준이 돼.
## 왜 중요한가
중요한 건 발표문에선 성능 숫자가 앞에 나오지만, 실제 도입은 컨텍스트·출력 한도·지원 API·가격표에서 갈린다는 점이야. 같은 OpenAI 모델이어도 여기 값이 달라지면 추천 답이 완전히 바뀐다. 그래서 이 페이지는 "얼마나 똑똑한가"보다 "우리 제품에 붙일 수 있는가"를 판단하는 용도로 읽는 편이 맞아.
