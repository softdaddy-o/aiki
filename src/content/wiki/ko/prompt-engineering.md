## 한 줄 정의
Prompt Engineering는 프롬프트 엔지니어링은 생성 인공 지능(GenAI) 모델에서 지정된 출력을 생성하기 위해 자연어 입력을 구조화하는 프로세스입니다. 컨텍스트 엔지니어링은 메타데이터, API 도구, 토큰 등 GenAI 모델에 제공되는 비프롬프트 컨텍스트 관리에 초점을 맞춘 소프트웨어 엔지니어링 관련 영역입니다라는 맥락에서 자주 언급된다.
## 어떻게 작동하나
클로드 API 문서라는 설명을 함께 보면, Prompt Engineering가 실제 제품과 연구 흐름에서 어떻게 쓰이는지 감이 잡힌다.
## 왜 지금 중요하나
Prompt Engineering는 최근 AI 제품, 모델, 워크플로를 읽을 때 기본 맥락을 잡아주는 용어다.
